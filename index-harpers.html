802.1Q
A port is untagged on one VLAN but tagged on many
Applied in: Lab LLD (VLAN/subnet plan) · Session Log §VLAN build · Lab Phase 1 exec · Onboarding §EXOS setup  —  IEEE 802.1Q deep-dive ↗
DHCP
The switch is the DHCP server — gateway, DNS, and range configured on the switch itself
Applied in: Lab Phase 1 (SW1 DHCP) · Lab Phase 2 (SW2 DHCP) · Session Log §VLAN config · Onboarding §Command ref  —  RFC 2131 deep-dive ↗
ARP

PoE
Power and data on the same cable — switch negotiates class with the AP

CAPWAP
