Appendix — Primary Sources & Further Reading

📚 Foundational Books

  • Computer Networking: A Top-Down Approach Kurose & Ross — the standard university textbook; Chapters 4–5 cover IP routing and link-layer exactly as we debugged it
  • TCP/IP Illustrated, Vol. 1 W. Richard Stevens — the definitive deep-dive on ARP, ICMP, IP routing, DHCP behaviour
  • Computer Networks Andrew Tanenbaum & David Wetherall — rigorous OSI-model treatment, 802.11 and VLANs well covered
  • Routing TCP/IP, Vol. 1 Jeff Doyle (Cisco Press) — inter-VLAN routing, static routes, and OSPF from a practitioner's view
  • CWNA: Certified Wireless Network Administrator Coleman & Westcott — 802.11 mechanics, CAPWAP, WPA3, enterprise WLAN design

🏫 University Courses & Lecture Notes

  • MIT 6.829 — Computer Networks (OCW) MIT OpenCourseWare — routing algorithms, TCP congestion control, network architecture; free lecture notes and problem sets
  • Stanford CS144 — Introduction to Computer Networking Stanford — hands-on TCP/IP implementation; lab assignments build a TCP stack from scratch
  • UMass COMPSCI 453 Lecture Slides Kurose's own slides for the textbook — excellent visual treatment of DHCP, NAT, and routing tables
  • Stanford EE384Y — Packet Switch Architectures Stanford EE — switching fabrics, VLAN forwarding, QoS; aligned with 5320 hardware internals

📄 Standards Bodies

  • IETF RFC Editor Canonical source for all IETF RFCs — IP, TCP, DHCP, DNS, NAT, SSH, CAPWAP, ZTP
  • IEEE 802 LAN/MAN Standards IEEE Xplore — 802.3 (Ethernet/PoE), 802.1Q (VLAN), 802.1D (STP), 802.11 (WiFi) — free to browse titles
  • Broadband Forum Technical Specs TR-069 (CPE management), TR-181 (device data model) — relevant to Quantum Fiber CPE behavior
  • Wi-Fi Alliance — Wi-Fi 6 (802.11ax) Certification program and technical overview for the standard underpinning AP3000 wireless

🌐 Vendor & Practitioner Docs

  • Extreme EXOS 33.5 User Guide Authoritative source for every EXOS CLI command used — VLAN, DHCP, ipforwarding, iqagent
  • Extreme XIQ Cloud Documentation XIQ Switch Policy, AP adoption, SSID-to-VLAN mapping, and IQ Agent configuration
  • Cloudflare Learning Center — Routing & NAT Cloudflare's free reference — excellent plain-English treatment of NAT, DNS, BGP, and IP routing
  • Cisco LAN Switching Technology Cisco's switching reference (VLAN, STP, inter-VLAN routing) — vendor-agnostic concepts apply directly to EXOS
  • PacketLife Cheat Sheets Handy one-page references for subnetting, 802.1Q, STP timers, and TCP/IP headers
